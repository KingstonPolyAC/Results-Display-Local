[How to Run]
Make sure you have python installed in your system from https://www.python.org/downloads/release/python-3110/

# Extract the project.zip file in any directory
#Open a CMD window in the project directory and then follow the instructions below

# Create a virtual environment 
python -m venv .venv

# Activate the virtual environment
## Windows
.venv\Scripts\activate
## macOS and Linux
source .venv/bin/activate

# Install Packages
pip install -r requirements.txt


# Run the executable file "Launch Results Display.exe" from the project directory
